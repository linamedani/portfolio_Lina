Java Eclipse Travail d'équipe

Projet en Java développé en groupe. Application "Ardoise Magique" avec interface graphique, classes abstraites, héritage, interface, et gestion d’événements.

Ce projet a été réalisé en groupe dans le cadre d’un exercice sur la programmation orientée objet en Java.

L’objectif était de créer une ardoise interactive permettant de dessiner à l’écran avec divers outils. Nous avons travaillé sur les spécifications ensemble pour bien définir les fonctionnalités clés à intégrer.

Le projet a été codé sous Eclipse, en utilisant des classes abstraites, l’héritage et des interfaces. L’interface graphique permettait d’interagir avec la souris pour dessiner, effacer ou changer d’outil.

Une bonne organisation d’équipe et des échanges réguliers ont été essentiels pour structurer le travail et livrer une application fonctionnelle.
